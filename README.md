# Palmer Penguins MLOps Pipeline with EDA and Explainable AI

This repository implements an end‑to‑end MLOps workflow for the **Palmer Penguins** dataset from OpenML (id: 42585), a multiclass classification task for penguin species.

The project showcases:

- Reproducible **data and model versioning** with DVC  
- **Hyperparameter optimization** with Optuna and experiment tracking with MLflow  
- **Model serving** via BentoML with a JSON API  
- **Feature engineering** (a biologically‑motivated `bill_body_ratio` feature)  
- Additional **data science** components: EDA, learning curves, SHAP‑based explainability, simple EDA report
